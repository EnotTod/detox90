# 90 дней свободы

Минималистичное PWA-приложение для трекинга отказа от вредных привычек — без регистрации, без сервера, без рекламы.

**[Открыть приложение →](https://enotTod.github.io/detox90)**

---

## Что это

Трекер на 90 дней для одновременного отказа от курения и других зависимостей. Каждый день показывает что именно изменилось в твоём организме — конкретно, по системам тела.

## Возможности

- **День за днём** — детальный статус для 15 ключевых точек (1, 2, 3, 7, 14, 21, 30, 45, 60, 75, 90)
- **Что изменилось** — 4 карточки по системам: лёгкие, мозг, сердце, гормоны, кожа
- **Что потеряешь при срыве** — конкретные потери показываются перед подтверждением
- **11 ачивок** — разблокируются по дням
- **Рекорд и счётчик попыток** — система памяти между сессиями
- **Работает офлайн** — всё хранится в браузере, никаких серверов

## Установка на iPhone

1. Открой в Safari: `https://enotTod.github.io/detox90`
2. Нажми **Поделиться** → **На экран «Домой»**
3. Готово — работает как нативное приложение

Чистый HTML + CSS + JS. Никаких зависимостей. Один файл.
